Bootstrap Cloudinary upload

A javascript functionality which helps you to accelerate the integration of Cloudinary in any new Bootstrap project.
This sample works upon the Bootstrap 4 Boilerplate.

The uploaded file url will be "saved" to the "cloudinaryLoadedImg" hidden input. Based on the "pure JS Cloudinary sample" at codepen
